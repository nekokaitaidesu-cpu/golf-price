# Project X Rifle Flighted 6.0（アイアン用スチールシャフト）

画像から特定したシャフトの調査メモ。

## スペック
- 名称: Project X Rifle **Flighted** 6.0（Precision / 旧 Rifle 製、Made in USA）
- 種類: アイアン用スチールシャフト
- フレックス: 6.0 = S（ステフィ）相当
- 重量: 約125g（5番アイアン基準）
- ライン: 5.0 / 5.5 / 6.0 / 6.5 / 7.0（約115〜135g）
- 特徴: ステップレス設計・インターナルライフリング
- 適正HS目安: 約90〜105 MPH（アイアンで約40〜47m/s前後）

## Flighted とは
番手ごとにキックポイントを変える設計。
- ロングアイアン → 低キックポイントで上がりやすく高弾道
- ショートアイアン → 高キックポイントで低弾道・方向性重視
セット全体で弾道を最適化する。

## 向いている人
中〜上級者・アスリート系で、HSそこそこ速め。低スピンで強い弾道を好み、吹け上がり/ばらつきを抑えたい人。
向きにくい人: HS遅め・非力・球が上がりにくい人（軽量スチールやカーボン推奨）。
